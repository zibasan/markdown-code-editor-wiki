Markdown Editorは、[VSCode](https://code.visualstudio.com/ "VSCodeのページにジャンプする")ライクの、
**Markdownファイル**の編集に特化したエディターです。

:::note
Markdownとは、以下のような構文で書かれた軽量マークアップ言語です。
```markdown title="Markdown_Syntax.md"
**こんにちは**
*あいうえおかきくけこさしすせそ*
---
# 見出し1
## 見出し2
```
:::

## 始めましょう

Markdown Editorは**Github Pagesでホストされたブラウザアプリ**と、**PCにインストールしたデスクトップアプリ**で使用できます。
:::caution
- ブラウザアプリとデスクトップアプリでは一部仕様が異なります。
  - 基本的にデスクトップアプリを優先して開発しているため、デスクトップアプリの使用をおすすめします。
- ブラウザアプリをモバイル端末で開くことは非推奨です。
:::

<details>
<summary>ブラウザアプリで使う</summary>

ブラウザアプリはGithub Pagesでホストされた[Webページ](https://zibasan.github.io/markdown-editor/)から使用することができます。

</details>

<details>
<summary>デスクトップアプリで使う</summary>

デスクトップアプリは、[Github上のMarkdown EditorのリポジトリにあるReleases](https://github.com/zibasan/markdown-editor/releases)からインストールして使用することができます。

</details>

### デスクトップアプリで使う場合

Markdown Editorのデスクトップアプリは、**WindowsとmacOSの両方**に対応しています。

#### Windowsの場合

`Markdown-Editor-Setup-<version>.exe`を使ってインストールできます。

#### macOSの場合

`Markdown-Editor-<version>-arm64.dmg`を使ってインストールできます。

:::caution
macOSでインストーラーを開こうとすると、`開発元を検証できないため開けません`ダイアログが表示されます。

このダイアログは以下の手順でバイパスできます。

1. Markdown EditorのインストーラーをFinderに配置する。
2. `Control ^`キーを押しながらアイコンをクリックし、`開く`をクリックする。
3. 再度`開く`をクリックする。
:::
